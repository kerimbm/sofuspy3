Extensions mathématiques de Blockly : organisation des fichiers

Le fichier de configuration permettant de modifier la taille des menus et des bulles d'aide est le fichier css_accessible.js.

Le dossier "keys"

Le dossier "keys" contient les librairies standard de Blockly, hors extensions.

Le dossier "utils"

Le dossier "utils" contient divers utilitaires : redirection de l'affichage en bas de page, pliage/dépliage du programme Blockly...

Le dossier "editeurs"

Le dossier "editeurs" regroupe les librairies permettant d'utiliser l'éditeur Ace ainsi que l'interpréteur python Skulpt.

Le dossier "sofuspy"

Spécifique à l'extension SofusPy.

Le dossier "xcas"

Spécifique à l'extension "tableur Xcas".

Le dossier "r"

Spécifique à l'extension "statistiques avec R".
